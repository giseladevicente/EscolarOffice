# EscolarOffice – Proyecto Front-End

Proyecto desarrollado como Proyecto Final de **Front-End JavaScript** dentro del programa **Talento Tech**.

## Descripción

**EscolarOffice** es una página web ficticia de venta de artículos escolares, comerciales y artísticos.
Fue desarrollada utilizando **HTML semántico**, **CSS puro**, **Bootstrap** y **JavaScript** para la estructura, el estilo visual y la interacción con el usuario.

El diseño es **responsive** y sigue la metodología **mobile-first**, utilizando **Flexbox**, **CSS Grid** y **Media Queries** para adaptarse a distintos tamaños de pantalla.


### Breakpoints utilizados

- **Extra small devices:** 400px x 600px
- **Small devices:** 601px x 400px
- **Medium devices:** 768px x 600px
- **Large devices:** 992px x 800px
- **Extra large devices:** 1200px x 800px

La sección **Contacto** incorpora un formulario funcional configurado con **Formspree** para gestionar el envío de datos sin necesidad de un backend adicional.

---

## Características

- Carrito de compras dinámico
- Persistencia de productos usando localStorage
- Diseño responsive
- Catálogo de productos por categorías
- Buscador visual en el header
- Página de reseñas de clientes
- Formulario de contacto funcional
- Carousel en la página principal
